# SvelteDocs

A small, reusable Svelte documentation engine. This sample content ships with
the repo so `npm run demo` renders something real, offline, with no external
content repo.

## What you're looking at

This page is built by the `svelte-docs-build` toolkit in `--local` mode from
`demo/sample-content/`, then rendered client-side by the `<Docs>` engine.

## Features

- Sidebar navigation with collapsible categories
- Table-of-contents rail with scroll-spy
- Full-text search (FlexSearch), lazy-loaded
- Version + language selectors
- Light/dark theme via the host's CSS variables
